# PI. Incorporación de recursos multimedia optimizados para la Web (audio y video)

Una página web diseñada para recrear la atmósfera visual y sonora inspirada en *Silent Hill f*, utilizando una estética basada en tonos rojos, negros y texturas granuladas.  
El objetivo es transmitir una sensación inquietante y minimalista, mientras se presentan elementos multimedia integrados de forma limpia y responsiva.

---

## Descripción del Proyecto

El proyecto consiste en una página temática que incorpora dos pistas de audio y un fragmento de video.  
Toda la interfaz está construida sobre una base visual oscura, con detalles en rojo que evocan la floración característica de *Silent Hill f*.

La página presenta:

- Dos archivos de audio ubicados en la carpeta `SHf`
- Un archivo de video con reproducción controlada
- Tarjetas visuales que acompañan el contenido multimedia
- Fondo con texturas y efectos sutiles
- Estilo minimalista orientado al terror atmosférico
- Diseño responsivo para adaptarse a cualquier pantalla

---

## Objetivo

Crear una experiencia digital que combine diseño, sonido y video para capturar la esencia visual y emocional de *Silent Hill f*, enfocándose en un ambiente oscuro sin sacrificar claridad visual o funcionalidad.

---

## Contenido Multimedia

Los archivos utilizados se encuentran en la carpeta `SHf`:

- `SHf2.mp3`
- `SHf33.mp3`
- `SHf.mp4`

Cada uno se presenta en tarjetas con su respectivo encabezado y descripción para contextualizar su función dentro del ambiente narrativo.

---

## Diseño y Estética

La estética del proyecto se construye sobre:

- Paleta negra y roja con alto contraste
- Textura granulada simulando ruido visual
- Tipografía monoespaciada en elementos clave
- Uso de sombras y luces tenues para generar profundidad
- Tarjetas de medios con iluminación suave y bordes redondeados
- Uso de gradientes y capas de mezcla (`mix-blend-mode`) para el fondo

Este diseño refleja la sensación silenciosa, incómoda y atmosférica típica de la saga.

## Responsividad

El sitio fue construido usando:

- CSS Grid
- Unidades flexibles
- Breakpoints para pantallas pequeñas

Esto garantiza una correcta visualización en:

- Computadoras de escritorio  
- Tablets  
- Dispositivos móviles  

---

## Tecnologías Utilizadas

- HTML5  
- CSS3 (gradientes, blur, grid, tipografías monoespaciadas, efectos visuales)  
- Reproducción nativa de audio y video  
- No se utiliza JavaScript, manteniendo un diseño minimalista y limpio
